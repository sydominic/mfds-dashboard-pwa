Streamlit Cloud Secrets 입력 예시

1) Supabase 연결 필수값

DATABASE_URL = "postgresql://postgres.xxxxxx:YOUR_PASSWORD@aws-0-ap-northeast-2.pooler.supabase.com:6543/postgres"

2) 선택값: 접속 시 1일 1회 자동 수집 여부

AUTO_COLLECT_ON_LOAD = false

권장:
- 처음에는 false 유지
- 앱 접속 속도를 빠르게 유지하려면 false
- 자동 수집을 켜고 싶으면 true로 변경


권장 운영:
AUTO_COLLECT_ON_LOAD = false 로 두고, 필요할 때만 수동 재수집을 누르세요.
최근 7일/14일은 빠른 수집 모드로 동작합니다.


v3 참고:
- AUTO_COLLECT_ON_LOAD = false 권장
- 조회 버튼은 DB 조회만 수행합니다.
- 빠른수집/기간수집 버튼을 눌렀을 때만 식약처 사이트 수집이 실행됩니다.


v4 참고:
- Supabase 연결이 안 되면 앱 화면에 DATABASE_URL 점검 안내가 표시됩니다.
- connection string의 postgres.xxxxxx는 예시가 아니라 Supabase가 제공한 실제 값이어야 합니다.


v5 참고:
- 빠른수집 후 [게시판별 수집 결과 보기]에서 m_74, m_99 등 각 구분별 확인 건수를 확인하세요.
- 최신 글이 누락되면 해당 게시판의 확인 건수와 최신게시일을 먼저 확인하면 됩니다.
