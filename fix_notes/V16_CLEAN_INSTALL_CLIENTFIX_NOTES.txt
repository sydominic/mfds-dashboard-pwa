V16 반영사항

원인:
- 이전 zip에 Linux 환경에서 생성된 client/node_modules가 포함될 수 있었습니다.
- Windows에서는 node_modules/.bin/vite.cmd가 있어야 npm run dev에서 vite를 인식합니다.
- 폴더만 보고 설치를 건너뛰면서 'vite is not recognized' 오류가 발생했습니다.

수정:
1. 배포 zip에서 client/node_modules, server/node_modules, client/dist 제외
2. run_local_core.bat가 client/node_modules/.bin/vite.cmd 존재 여부를 확인
3. vite.cmd가 없으면 기존 client/node_modules를 삭제 후 npm install 재수행
4. start_client.bat도 같은 검사를 수행
5. start_client.bat의 중복 --host/--port 인자 제거
6. UI/API version: v16-clean-install-clientfix
